# Xbox Dino Shooter

一个使用 Python 和 pygame 制作的 Xbox 手柄横版跑酷射击小游戏。

## 运行环境

- Python 3.10+
- pygame

## 安装依赖

```powershell
pip install -r requirements.txt
```

## 启动游戏

```powershell
python dino_game.py
```

## Xbox 手柄按键

- A / Button 0：按住下蹲
- Y / Button 3：跳跃
- B / Button 1：射击

## 素材文件

请确保以下图片和 `dino_game.py` 位于同一目录：

- `robotman.png`
- `cactus.png`
- `bird.png`
- `jetflight.png`
